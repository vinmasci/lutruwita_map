## Prerequisites
```bash
- Node.js 18+
- PostgreSQL 14+ with PostGIS
- MongoDB 6+
- pnpm
```

## Quick Start
```bash
git clone https://github.com/vinmasci/lutruwita_map.git
cd lutruwita_map
pnpm install
pnpm dev
```
